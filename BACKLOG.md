- **LanguageTool grammar pass** — `src/lib/language-tool.ts` calls LanguageTool's public API (`api.languagetool.org`) and auto-applies high-confidence TYPOS/GRAMMAR/PUNCTUATION/CASING/TYPOGRAPHY fixes.
